ENDPOINTS SUMMARY
=================

PUBLIC (No Auth Required)
--------------------------
GET /l/[code]                          → 302 redirect + click tracking


AUTHENTICATED (requires client/admin role + orgId scope)
---------------------------------------------------------
GET    /api/v1/links                   → List links (paginated)
POST   /api/v1/links                   → Create new link
GET    /api/v1/links/[id]              → Get link + stats
DELETE /api/v1/links/[id]              → Delete link
GET    /api/v1/links/[id]/stats        → Get analytics only


UI ROUTES
=========
/portal/social/links                   → Link management interface

KEY DESIGN PATTERNS
===================

1. SHORT CODE GENERATION
   → 7-char alphanumeric (A-Z a-z 0-9)
   → Org-scoped uniqueness
   → Auto-regenerate on collision (up to 10 attempts)

2. UTM PARAMETER HANDLING
   → Store in link document (immutable)
   → Append on redirect (server-side)
   → Live preview in UI

3. CLICK TRACKING
   → Fire-and-forget (non-blocking)
   → Capture: timestamp, referrer, user agent, IP
   → Increment counter on link document

4. ANALYTICS
   → In-memory aggregation from last 1000 clicks
   → Daily breakdown, top referrers, top countries
   → Fast queries for typical use (<1s)

5. MULTI-TENANCY
   → All operations scoped to orgId
   → Auth enforces org isolation
   → Different orgs can't access each other's links

6. PUBLIC REDIRECTS
   → No auth required (/l/[code])
   → Format validation (alphanumeric)
   → Fallback to homepage if not found

CODE EXAMPLES
=============

Create a link programmatically:
────────────────────────────────
import { createShortLink } from '@/lib/links/shorten'

const link = await createShortLink(
  orgId,
  'https://example.com/article',
  {
    utmSource: 'twitter',
    utmCampaign: 'launch'
  },
  userId
)

console.log(link.shortUrl) // https://app.com/l/xYz9AbC


Create a link via API:
──────────────────────
const res = await fetch('/api/v1/links', {
  method: 'POST',
  body: JSON.stringify({
    originalUrl: 'https://example.com',
    utmSource: 'email'
  })
})
const { data } = await res.json()


List links:
───────────
const res = await fetch('/api/v1/links?page=1&limit=20')
const { data, meta } = await res.json()


Get link with stats:
────────────────────
const res = await fetch('/api/v1/links/abc123')
const { data } = await res.json()
console.log(data.stats.totalClicks)


Get analytics only:
───────────────────
const res = await fetch('/api/v1/links/abc123/stats')
const { data } = await res.json()


Delete a link:
──────────────
await fetch('/api/v1/links/abc123', { method: 'DELETE' })


Access short link (browser):
─────────────────────────────
https://app.com/l/xYz9AbC
→ 302 redirect to original URL with utm params
