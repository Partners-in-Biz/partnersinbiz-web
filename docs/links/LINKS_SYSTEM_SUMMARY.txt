================================================================================
DATABASE SCHEMA
================================================================================

Collection: shortened_links
├── id (String) — Firestore document ID
├── orgId (String) — Organization ID
├── originalUrl (String) — Full destination URL
├── shortCode (String) — 7-char code (e.g., xYz9AbC)
├── shortUrl (String) — Generated short URL
├── utmSource (String, optional) — UTM source param
├── utmMedium (String, optional) — UTM medium param
├── utmCampaign (String, optional) — UTM campaign param
├── utmTerm (String, optional) — UTM term param
├── utmContent (String, optional) — UTM content param
├── clickCount (Number) — Total clicks
├── createdBy (String) — User ID who created
├── createdAt (Timestamp) — Creation time
├── updatedAt (Timestamp) — Last update time
└── clicks/ (Subcollection)
    ├── id (String) — Click document ID
    ├── linkId (String) — Parent link ID
    ├── orgId (String) — Organization ID
    ├── timestamp (Timestamp) — Click time
    ├── referrer (String, null) — Referrer URL
    ├── userAgent (String, null) — User agent string
    ├── ip (String, null) — Client IP address
    └── country (String, null) — Inferred country (TODO)

================================================================================
API REFERENCE
================================================================================

GET /api/v1/links
  Query: page=1&limit=20
  Response: { success, data: ShortenedLink[], meta: { total, page, limit } }
  Auth: client/admin

POST /api/v1/links
  Body: { originalUrl, utmSource?, utmMedium?, utmCampaign?, utmTerm?, utmContent? }
  Response: { success, data: ShortenedLink }
  Auth: client/admin

GET /api/v1/links/[id]
  Response: { success, data: ShortenedLink & { stats: LinkStats } }
  Auth: client/admin

DELETE /api/v1/links/[id]
  Response: { success, data: { id, deleted: true } }
  Auth: client/admin

GET /api/v1/links/[id]/stats
  Response: { success, data: { id, shortUrl, originalUrl, stats: LinkStats } }
  Auth: client/admin

GET /l/[code]
  Response: 302 redirect to original URL with UTM params
  Auth: None (public)

================================================================================
SECURITY & MULTI-TENANCY
================================================================================

✓ Multi-tenant isolation enforced
  - All queries scoped to orgId
  - Org from authenticated user context
  - Different orgs cannot access each other's links

✓ Authentication enforced
  - withAuth('client') on all write endpoints
  - withTenant() to resolve and scope orgId
  - Public redirects (/l/[code]) have no auth

✓ URL validation
  - URL constructor validates format
  - Prevents invalid/malicious URLs
  - UTM params stored, not passed via query string

✓ Click tracking security
  - Only non-sensitive data captured
  - No personal data stored
  - IP addresses only for geolocation (future)
