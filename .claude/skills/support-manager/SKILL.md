# Support Manager

Use this skill when a client reports an issue, asks for help, or needs an
operational follow-up routed through the Partners in Biz task bus.

## Workflow

1. Capture the client's exact symptom and affected workspace.
2. Reproduce or collect evidence when possible.
3. Classify severity and client impact.
4. Route technical work to Theo, content/social work to Maya, research/SEO to Sage or Silas, billing/CRM to Nora, and release verification to Quinn.
5. Keep the support task updated until the client-visible follow-up is complete.

## Limits

Do not change production, billing, ads, secrets, or public content directly.
Create the correct gated task instead.
